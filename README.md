# Prasad Kadu — Portfolio Website

Personal portfolio and resume website for Prasad Kadu, a Full Stack Software Developer.

## Live Site

[kaduprasad.github.io](https://kaduprasad.github.io)

## Tech Stack

- Pure HTML5, CSS3, Vanilla JS — no frameworks or build tools
- Custom dark theme with animated background and scroll-reveal effects
- Devicon & Font Awesome for skill/tech icons
- Google Fonts (Inter)
- Deployed via GitHub Pages

## Sections

- **About** — Introduction, contact info, social links
- **Experience** — Work history with timeline
- **Education** — Academic background with institution logos
- **Projects** — Enterprise and personal projects
- **Skills** — Tech stack with icon showcase
- **Certifications** — Microsoft certifications
- **Interests** — Hobbies and activities

## Local Development

No build step needed. Open `index.html` directly in a browser or use a local server:

```bash
npx serve .
```
